# QAlite Frontend

Aplicação frontend em React + Vite para gestão de cenários, suítes e ambientes de testes.

## Requisitos

- Node.js 20+
- npm 10+

## Scripts

- `npm run dev`: ambiente local
- `npm run build`: build de produção
- `npm run typecheck`: checagem TypeScript
- `npm run lint`: lint do projeto

## Estrutura inicial

- `src/presentation`: páginas, componentes e contextos
- `src/domain`: entidades e contratos de repositório
- `src/infrastructure`: integrações, serviços e acesso a dados
- `src/shared`: utilitários e configurações comuns
